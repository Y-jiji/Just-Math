## 字面值类型 [SYM]
::::文字::::
在 $\LaTeX$ 标记语言中, 字面值能够以角标,上下标等方式进行组合. 
此处为字面值也构造一个类型系统, 便于定义模式. 

## 配对符号 [SYM]
::::文字::::
以下四种符号的配对被称为配对符号. 
| 左   | 右         |
| ---- | ---------- |
| $[$  | $]$ 或 $)$ |
| $($  | $]$ 或 $)$ |
| $\{$ | $\}$       |
| $\|$ | $\}$       |
| $\big<$     |  $\big>$          |
一个模式中不会含有未配对的配对符号. 
::::注记::::
$[$ 和 $)$ 的混用主要是为了照顾分析中的区间. 至于物理当中的 $\big<a|b|c\big>$ 是完全不打算使用的.
